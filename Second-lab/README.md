# Анализ и классификация данных Iris с помощью машинного обучения

## Цель работы

Создать полный пайплайн машинного обучения. В качестве данных используется классический набор данных Iris, задача – классификация цветков на три различных вида.

## Описание набора данных

Набор данных Iris содержит 150 записей, каждая из которых описывает цветок по 4 признакам: длина и ширина чашелистика, длина и ширина лепестка. Метка класса представляет собой вид цветка (Setosa, Versicolor, Virginica).

## Задача

Обучить модель на основе алгоритма машинного обучения (например, логистическая регрессия или SVM) для классификации вида цветка. Оценить качество модели, используя метрику точности (accuracy) на тестовой выборке.

## Дополнительные задания

1. Попробуйте другие алгоритмы классификации, такие как SVM или K-ближайших соседей (KNN), и сравните результаты.
2. Используйте кросс-валидацию для более точной оценки производительности модели.

# Analysis and Classification of Iris Data Using Machine Learning

## Goal

Create a complete machine learning pipeline. The data used is the classic Iris dataset, with the task of classifying flowers into three different types.

## Dataset Description

The Iris dataset contains 150 records, each describing a flower by 4 features: sepal length and width, petal length and width. The class label represents the type of flower (Setosa, Versicolor, Virginica).

## Task

Train a model based on a machine learning algorithm (e.g., logistic regression or SVM) to classify the type of flower. Evaluate the model's quality using the accuracy metric on the test set.

## Additional Tasks

1. Try other classification algorithms, such as SVM or K-nearest neighbors (KNN), and compare the results.
2. Use cross-validation for a more accurate assessment of the model's performance.
